# 🚗 Car Sales Analysis Dashboard

## 📌 Project Overview
The Car Sales Analysis Dashboard is an interactive Power BI dashboard designed to analyze automobile sales performance across multiple car brands. It provides valuable insights into sales, revenue, market share, and top-performing models through intuitive visualizations and key performance indicators (KPIs).

## 🎯 Objectives
- Analyze overall car sales performance.
- Compare revenue across different car brands.
- Monitor market share distribution.
- Identify top-performing vehicle models.
- Track sales trends over time.
- Support data-driven business decisions.

## 📊 Dashboard Features
- Total Units Sold Analysis
- Total Revenue Tracking
- Brand-wise Market Value Comparison
- Market Share Analysis
- Top 5 Models by Revenue
- Year-wise Sales Trend Visualization
- Interactive Brand Filters and Slicers

## 🛠️ Tools & Technologies
- Power BI
- Microsoft Excel / CSV Dataset
- Data Modeling
- Data Visualization
- DAX

## 📈 Key Metrics
- Units Sold: 73K
- Total Revenue: 245M
- Vehicle Variants: 24
